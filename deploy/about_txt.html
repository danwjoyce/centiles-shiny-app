About / Warning

This is an untested beta version (i.e. for experimenting with) and it has not been validated against charts used in clinical practice. It should not be relied upon as the sole means of calculating centiles for height or blood pressure in routine clinical practice.

Operation is self-explanatory - input the child's parameters and the centiles get calculated

To view current data on a centile plot (in the Plot Tab), select the required plot in the drop-down box, and hit the ?Plot? button.

The Summary tab contains a text report for copying and pasting into letters, notes etc.

Created using R, ggplot2, reshape and grid packages running in Shiny. I am happy to make the code freely available (e.g. if you want to port it to a more flexible platform, run it locally on an R/Shiny server)

References / Sources

The current version supports graphing and centile calculations using only US-based CDC2000 parameters. The UK versions are forthcoming.

Kuczmarski RJ, Ogden CL, Guo SS, et al. (2000) CDC growth charts for the United States: Methods and development. National Center for Health Statistics, Vital Health Stat. 11 (246).


Similarly, BP calculations are derived from :

National High Blood Pressure Education Program Working Group on High Blood Pressure in Children and Adolescents, (2004) The Fourth Report on the Diagnosis, Evaluation, and Treatment of High Blood Pressure in Children and Adolescents. Pediatrics, Vol. 114 No. Supplement 2 August 1, pp. 555-576.
